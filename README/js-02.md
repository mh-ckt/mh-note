## 移动端 click 事件的 300ms 延迟如何解决？

1. 移动端点击有 300ms 的延迟是为了兼容双击缩放的功能设计的，如果不需要双击缩放可以在 mata 标签中添加 user-scalable=no 用于禁止缩放。

```html
<meta
  name="viewport"
  content="width=device-width, initial-scale=1, user-scalable=no"
/>
```

2. 添加 touch-action:manipulation; 告诉浏览器此元素不需要双击缩放。

```css
button,
a,
[role='button'] {
  touch-action: manipulation; /* 禁用双击缩放 */
}
```

## js 加载执行：async,defer,preload,prefetch 的区别？

async：立即下载（异步，不会阻碍文档解析），异步执行（执行的时候会阻碍文档解析）
defer：立即下载（异步，不会阻碍文档解析），延迟执行，在整个页面都解析完毕后执行
preload：提前下载，需要的时候立即执行，无需再下载
prefetch：提前下载，在未来的某个页面可能会执行，节省下载时间
