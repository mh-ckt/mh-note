## 2024/11/12/14:00 申通快递面试（外派上海，线上）薪资 15K-18k

1. `let str = 'hello world'`统计上面字符串中字符出现的次数，统计后格式如下：`{h: 1, e: 1, l: 3, o: 2, w: 1, …}`

   ```js
   let str = 'hello world'
   let obj = str
     .replace(/\s+/g, '') //  \s+指的是匹配一个或者多个空格
     .split('')
     .reduce((pre, item, index, arr) => {
       if (arr.indexOf(item) === index) {
         pre[item] = 1
       } else {
         pre[item]++
       }
       return pre
     }, {})
   console.log(obj)
   ```
